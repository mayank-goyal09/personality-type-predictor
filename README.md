### Prerequisites

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/mayank-goyal09/personality-type-predictor.git
cd personality-type-predictor
```

2. **Install dependencies**


3. **Train the model** (if not already trained)
